Start

Start using Defresh for free with 1 line of code. Paste this into the <head> section of your webpage:

<script src="https://defresh.glitch.me/defresh.js" async></script>


Or you can download the code and paste it into the root folder of your site. Note: your file MUST be named /defresh.js:

<script src="/defresh.js" async></script> Download Code


That's all you have to do because Defresh will do the rest! Also, remember, the script will only work if the following page has the script, too. Read more about how to use Defresh in Javascript at our tutorials page.
