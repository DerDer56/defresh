Tutorials

You can use defresh in your site's .js files with the defresh() function.
    The function has two parts:
  • The Page
  • The Method

    • You can use this in your javascript file by calling a function like this:

    defresh('/page', 'method'); Try It


    The '/page' can be any page on YOUR SITE, such as index.html, which will take you to yoursite.url/index.html. If you try to use a site like https://youtube.com/, it will not take you to youtube.com, but instead to yoursite.url/https://youtube.com/ and return a 404. 404.

      The 'method' part can be any of 3 options:
  • Push (you can use the back button to go back)
  • Defresh (you can use the back button to go back)
  • None (default) (the current url does not change but the content does)

    • Warning: if you use 'none', the page content will go away when you refresh the page. Also, we recently updated Defresh to work with dynamically added links.
